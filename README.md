# Follow the Rabbit 🐇

A one-page event site for an underground electronic music night.
No framework. No dependencies. No excuses.

**[→ Live site](https://rabbit.sonidosessions.es)**

---

## What's this?

Static site for a real event. Built fast, kept lean — vanilla HTML, CSS, and JS doing
exactly what they're supposed to do without a node_modules folder the size of a planet.

Features a DJ lineup carousel, photo gallery with lightbox, animated marquee, and a
CRT glitch effect because subtlety is overrated at a techno night.

## Stack

- HTML / CSS / JS — that's it
- Google Fonts (Inter + Space Grotesk)
- GitHub Actions → Hostinger deploy via SSH

No build step. No bundler. Open `index.html` and you're home.

## Security

Hardened before going public: CSP, HSTS, X-Frame-Options, EXIF-stripped images.
Probably more locked down than it needs to be for a rave website, but here we are.
